## Overview

SameChat is a Java-based chat application designed to provide seamless communication between users. Built with a focus on simplicity and reliability, SameChat offers a user-friendly interface for real-time messaging.

## Features

- 💬 Real-time messaging
- 👥 User-friendly interface
- 🔐 Secure communication
- 📱 Cross-platform support
- ⚡ Fast and responsive

## Tech Stack

- **Language**: Java
- **Architecture**: Modular and scalable design

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Maven or Gradle (depending on build setup)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Hamraj37/SameChat.git
cd SameChat
```

2. Build the project:
```bash
# Using Maven
mvn clean build

# Using Gradle
gradle build
```

3. Run the application:
```bash
java -jar SameChat.jar
```
